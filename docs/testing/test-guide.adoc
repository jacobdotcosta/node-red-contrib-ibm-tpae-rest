== Testing

https://github.com/node-red/node-red-node-test-helper


```bash
$ npm install node-red-node-test-helper node-red --save-dev
```

== Execute tests

Unit tests.

[source,bash]
----
$ npm run-script test
----

Integration tests.

[source,bash]
----
$ npm run-script test-it -- --tpaeHost=icd761.trikorasolutions.net --tpaePort=9080
----
